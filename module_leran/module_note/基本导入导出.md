## 模块的基本导出和导入

ES6中的模块导入和导出分为两种:
1. 基本导入导出
2. 默认导入导出

## 基本导出
 类似于 exports.xxx=xxxx
 基本导出可以有多个，每个必须有名称
 ``` js
  export 声明表达式
 ```
 或
  ``` js
  export {具名符号}
 ```
 由于基本导出必须具有名称，所以要求导出内容必须跟上**声明表达式或具名符号**
 ## 基本导入
 对于基本导出，如果进行导入，使用下面的代码
 ```js
    import {导入的符号列表}  from "模块路径"
 ```
注意以下细节：
1. 导入时，可以通过关键字 as 对导入的符号进行重命名
2. 导入时使用的符号时常量，不可修改
3. 可以使用*号导入所有的基本导出，形成一个对象
